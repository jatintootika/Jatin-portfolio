## 🚀 About The Project

This is a state-of-the-art interactive portfolio designed to showcase projects, skills, and experience in a modern, cinematic way. Featuring a live 3D Spline background, an integrated AI chatbot, and a terminal-style project viewer, this portfolio stands out from the standard web experience.

### ✨ Key Features

- **Cinematic 3D Hero**: Built with Spline and Framer Motion for a breathtaking first impression.
- **Interactive AI Chatbot (JatinBot)**: Integrated with a live Wikipedia API to answer real-world questions natively.
- **Terminal Project Viewer**: A retro-futuristic command-line interface to explore past projects.
- **Bento-Box About Section**: A highly modern grid layout to showcase statistics, skills, and experience.
- **Dynamic Skill Universe**: Orbiting skill icons in a 3D-like space.
- **Fully Responsive**: Optimized for desktops, tablets, and mobile devices.

## 💻 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your-username/portfolio.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the development server
   ```sh
   npm run dev
   ```
